# Exercise 2: 컨볼루션과 ReLU — 모범 답안

이 실습에서는 특성 추출의 첫 두 단계인 **컨볼루션**과 **ReLU**를 학습합니다.

---

## Q1. 커널 정의

3×3 커널을 정의합니다. (여러 가지 선택지가 있습니다)

```python
# 한 가지 예시
kernel = tf.constant([
    [-2, -1, 0],
    [-1, 1, 1],
    [0, 1, 2],
])
```

> **설명**:
> - 커널의 원소 합이 0~1 사이이면 밝기 유지
> - 이 커널은 **엠보싱(embossing)** 효과 생성
> - 다양한 선택이 가능 (edge detect, blur, sharpen 등)

---

## Q2. TensorFlow 컨볼루션 함수

컨볼루션을 수행하는 TensorFlow 백엔드 함수를 선택합니다.

```python
conv_fn = tf.nn.conv2d
```

> **설명**:
> | Keras 층 | TensorFlow 함수 | 용도 |
> |---------|-------------------|------|
> | `layers.Conv2D()` | `tf.nn.conv2d()` | 2D 이미지 컨볼루션 |
> | `layers.Conv1D()` | `tf.nn.conv1d()` | 1D 시계열 컨볼루션 |
> | `layers.Conv3D()` | `tf.nn.conv3d()` | 3D 비디오 컨볼루션 |

---

## Q3. ReLU 활성화 함수

특성을 검출하기 위해 ReLU 함수를 적용합니다.

```python
relu_fn = tf.nn.relu
```

> **설명**:
> - ReLU(Rectified Linear Unit): `max(0, x)`
> - 음수 값을 0으로 변환
> - 음의 특성을 제거하고 양의 특성만 활성화
> - Keras에서는 보통 `activation='relu'` 파라미터로 사용

---

## Q4. 수치 행렬에서의 컨볼루션

6×6 이미지와 2×2 커널로 컨볼루션 결과를 분석합니다.

**이미지:**
```
세로 선: 왼쪽 (0,1,0,0,0,0 패턴)
가로 선: 오른쪽 아래 (1,1,1)
```

**커널:**
```
[1, -1]
[1, -1]
```

**예상 효과:**
- 이 커널은 **경계선 검출** (특히 수평 경계)
- 원본이 1인 곳에서 음수 또는 양수 값 생성
- ReLU 후 양수만 살아남 → 가로선 경계 강조

**실제 결과:**
- 가로선 경계에서 2.0 (최대값)
- 다른 곳에서 0 또는 음수

## 핵심 개념

| 단계 | 함수 | 목적 |
|------|------|------|
| **Filtering** | `conv2d()` | 커널로 이미지 필터링 |
| **Detection** | `relu()` | 원하는 특성만 활성화 |
| **Condensing** | `max_pool()` | 특성 맵 압축 (다음 레슨) |

# 표준 커널 예제들
edge = [[-1, -1, -1],
