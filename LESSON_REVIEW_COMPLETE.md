## 🔍 주요 발견

### 1. 번역 방식의 이질성
- **수동 번역** (Lesson 1, 3, 6): 자연스럽고 구조 유지
- **자동 번역** (Lesson 2, 4, 5): 구조 손상, 오류 패턴 일치

### 2. 자동 번역 도구의 증거
- 들여쓰기 오류: 함수의 **첫 라인부터 일관되게** 발생
- 공백 오류: `.` 연산자 주변에서만 체계적으로 발생
- 수식 파괴: "2×2" → "2/×/2" (한자리 단위로 분해)

### 3. 공통 코드 블록의 오류 공유
```
set_seed() 함수와 데이터셋 로딩 코드가
Lesson 1, 5, 6에서 반복되며
각 레슨마다 다른 위치에서 오류 발생
→ 복사-붙여넣기 중 들여쓰기 손상의 증거
```
